# 荀子《勤學》學習指南

> 「學不可以已」——學習不能停止。戰國末期儒家代表人物荀子的核心教育思想，強調後天學習對改變人性、提升自我的重要性。

## 📚 作者與背景

### 荀子簡介
- **生卒**：約公元前313年－前238年
- **學派**：儒家代表人物，但思想融合法家元素
- **核心觀點**：「性惡論」——人性本惡，需通過後天學習來改變
- **歷史地位**：戰國末期最後一位儒家大師，韓非、李斯皆出其門下

### 寫作背景
- **時代**：戰國末期，社會動盪，百家爭鳴
- **目的**：勸導人們重視學習，修身養性
- **受眾**：士人、學子，尤其是追求進步的年輕人

## 第一段 <span class="dse-tag">必背段落</span>

### 原文

```
君子曰：學不可以已。青，取之於藍，而青於藍；冰，水為之，而寒於水。木直中繩，輮以為輪，其曲中規；雖有槁暴，不復挺者，輮使之然也。故木受繩則直，金就礪則利，君子博學而日參省乎己，則知明而行無過矣。
```

### 淺白講解

#### 一、白話翻譯
有修養的人說：**學習是不能停止的**。

靛青色是從藍草中提取的，卻比藍草顏色更深；冰是由水凝結而成的，卻比水更寒冷。

木材本來筆直得符合墨線標準，用火烤彎做成車輪後，它的弧度能符合圓規；即使後來曬乾變形，也不會再變直，這是因為火烤加工徹底改變了它。

所以木材經墨線量過就筆直，金屬刀劍在磨刀石上磨過就鋒利。君子廣泛學習，並且**每天多次反省檢討自己**，就能智慧明達、行為沒有過失。

#### 二、逐句深度解析

**1. 君子曰：學不可以已。**
- **字詞**：已，停止。
- **白話**：有修養的人說，學習絕對不能停止。
- **現代啟示**：學習是一輩子的事，不是階段性任務。在知識爆炸的時代，終身學習更是生存必需。

**2. 青，取之於藍，而青於藍；冰，水為之，而寒於水。**
- **字詞**：青，靛青染料；藍，藍草；為，凝結。
- **白話**：靛青從藍草提取卻更青，冰由水凝結卻更冷。
- **現代啟示**：**後天努力可以超越先天條件**。學生可以超越老師，今天的自己可以超越昨天的自己。這是對「長江後浪推前浪」的經典論述。

**3. 木直中繩，輮以為輪，其曲中規；雖有槁暴，不復挺者，輮使之然也。**
- **字詞**：中繩，符合墨線；輮，用火烤彎；槁暴，曬乾。
- **白話**：直木烤彎成車輪，即使曬乾也不會變直，因為加工改變了它。
- **現代啟示**：**教育、訓練、長期學習能徹底改變一個人**。好的習慣一旦養成就不易退回原狀，如同「習慣成自然」。

**4. 故木受繩則直，金就礪則利。**
- **字詞**：受繩，用墨線校正；就礪，在磨刀石上磨。
- **白話**：木材用墨線校正就直，刀劍磨過就鋒利。
- **現代啟示**：人也一樣，**需要規範和磨練才會進步**。放任不管只會越來越差，所謂「玉不琢，不成器」。

**5. 君子博學而日參省乎己，則知明而行無過矣。**
- **字詞**：博學，廣泛學習；參省，檢驗反省；知明，智慧明達。
- **白話**：君子廣泛學習且每天反省，就能明智而少犯錯。
- **現代啟示**：學習不只是吸收知識，還要**結合自我反省**，把學到的應用到自己身上，做到「知行合一」。

## 第二段 <span class="dse-tag">重要段落</span>

### 原文

```
吾嘗終日而思矣，不如須臾之所學也；吾嘗跂而望矣，不如登高之博見也。登高而招，臂非加長也，而見者遠；順風而呼，聲非加疾也，而聞者彰。假輿馬者，非利足也，而致千里；假舟楫者，非能水也，而絕江河。君子生非異也，善假於物也。
```

### 核心概念：善假於物

**「君子生非異也，善假於物也」**——君子的本性與一般人沒有差異，只是善於借助外物。

#### 逐句解析

**1. 吾嘗終日而思矣，不如須臾之所學也；吾嘗跂而望矣，不如登高之博見也。**
- **字詞**：須臾，片刻；跂，踮起腳尖。
- **白話**：整天思考不如學習片刻，踮腳遠望不如登高望遠。
- **現代啟示**：**學習比空想更有用**。與其自己苦思冥想，不如學習前人智慧；與其自己盲目摸索，不如站得高看得遠。

**2. 登高而招，臂非加長也，而見者遠；順風而呼，聲非加疾也，而聞者彰。**
- **字詞**：招，招手；彰，清楚。
- **白話**：登高招手看得遠，順風呼喊聽得清。
- **現代啟示**：**善用條件能事半功倍**。同樣的努力，如果方法得當、條件有利，效果會好很多。

**3. 假輿馬者，非利足也，而致千里；假舟楫者，非能水也，而絕江河。**
- **字詞**：假，借助；輿馬，車馬；楫，船槳；絕，橫渡。
- **白話**：藉助車馬能遠行，藉助船隻能渡河。
- **現代啟示**：**善用工具能彌補不足**。不會跑但會用車，不會游泳但會用船，同樣能達成目標。

**4. 君子生非異也，善假於物也。**
- **字詞**：生，通「性」，本性；假，借助。
- **白話**：君子的本性與一般人沒有差異，只是善於借助外物。
- **現代啟示**：**成功的關鍵不是天賦，而是善用資源**。聰明人不是天生比別人強，而是更會利用工具、資源、他人經驗。

#### 現代啟示

1. **善用工具**：現代學習要善用網絡、書籍、軟件等工具
2. **借助資源**：向老師請教、與同學討論、參加學習小組
3. **利用環境**：創造良好的學習環境，減少干擾
4. **學習方法**：找到適合自己的學習方法，提高效率

## 第三段 <span class="dse-tag">必背段落</span>

### 原文

```
積土成山，風雨興焉；積水成淵，蛟龍生焉；積善成德，而神明自得，聖心備焉。故不積蹞步，無以至千里；不積小流，無以成江海。騏驥一躍，不能十步；駑馬十駕，功在不舍。鍥而舍之，朽木不折；鍥而不舍，金石可鏤。螾無爪牙之利，筋骨之強，上食埃土，下飲黃泉，用心一也。蟹六跪而二螯，非蛇蟺之穴無可寄託者，用心躁也。
```

### 學習三要素

#### 逐句解析

**1. 積土成山，風雨興焉；積水成淵，蛟龍生焉；積善成德，而神明自得，聖心備焉。**
- **字詞**：興，興起；淵，深潭；神明，智慧；聖心，聖人的思想境界。
- **白話**：積土成山會有風雨，積水成淵會有蛟龍，積善成德會有智慧。
- **現代啟示**：**量變引起質變**。點滴積累最終會產生巨大變化，品德修養也是如此。每天進步一點點，長期會有巨大收穫。

**2. 故不積蹞步，無以至千里；不積小流，無以成江海。**
- **字詞**：蹞步，半步；無以，無法。
- **白話**：不積累小步不能遠行，不匯聚細流不能成海。
- **現代啟示**：**學習需要點滴積累**。沒有人能一步登天，所謂「不積小流，無以成江海」。

**3. 騏驥一躍，不能十步；駑馬十駕，功在不舍。**
- **字詞**：騏驥，千里馬；駑馬，劣馬；十駕，十天的行程。
- **白話**：千里馬跳一下不超過十步，劣馬走十天也能走遠，成功在於堅持。
- **現代啟示**：**堅持比天賦更重要**。聰明人不努力也沒用，普通人堅持不懈也能成功。這是「龜兔賽跑」的古代版本。

**4. 鍥而舍之，朽木不折；鍥而不舍，金石可鏤。**
- **字詞**：鍥，雕刻；舍，放棄；鏤，雕刻。
- **白話**：雕刻一下放棄，朽木也刻不斷；堅持雕刻，金石也能刻成功。
- **現代啟示**：**持之以恆是成功的關鍵**。再難的事，只要堅持就能成功；再簡單的事，放棄也做不成。

**5. 螾無爪牙之利，筋骨之強，上食埃土，下飲黃泉，用心一也。蟹六跪而二螯，非蛇蟺之穴無可寄託者，用心躁也。**
- **字詞**：螾，蚯蚓；黃泉，地下的泉水；跪，腳；螯，鉗子；蟺，通「鱔」，黃鱔。
- **白話**：蚯蚓條件差但專心，螃蟹條件好但浮躁。
- **現代啟示**：**專心致志比條件優越更重要**。條件不好但專心，能做成事；條件好但浮躁，一事無成。

## 📊 論證手法分析

### 1. 比喻論證（最突出的手法）
| 比喻 | 本體（學習） | 喻體 | 含義 |
|------|--------------|------|------|
| 青出於藍 | 學習的效果 | 靛青與藍草 | 後天學習能超越先天 |
| 冰寒於水 | 學習的提升 | 冰與水 | 學習能提高層次 |
| 木受繩則直 | 學習的規範 | 木材與墨線 | 學習需要標準指導 |
| 金就礪則利 | 學習的磨練 | 刀劍與磨石 | 學習需要實踐鍛煉 |
| 積土成山 | 學習的積累 | 土與山 | 學習需要長期堅持 |
| 鍥而不舍 | 學習的堅持 | 雕刻金石 | 學習需要持之以恆 |

### 2. 對比論證
- **青 vs 藍**：學習前後對比
- **冰 vs 水**：轉化前後對比
- **終日而思 vs 須臾所學**：空想與學習對比
- **騏驥一躍 vs 駑馬十駕**：天賦與堅持對比
- **蚯蚓 vs 螃蟹**：專心與浮躁對比

### 3. 舉例論證
- **登高而招**：善用條件
- **假輿馬、假舟楫**：善用工具
- **積土成山、積水成淵**：積累的效果

### 4. 歸納論證

### 作文示例

**題目：「談堅持」**

荀子在《勤學》中強調「鍥而不舍，金石可鏤」，又用「駑馬十駕，功在不舍」來說明即使資質平凡，只要堅持不懈，也能取得成功。這種堅持精神在今天的學習和生活中同樣重要……

**完整參考答案：**
荀子在《勤學》一文中，通過「鍥而不舍，金石可鏤」和「駑馬十駕，功在不舍」等經典論述，深刻闡明了堅持的重要性。在現代社會中，這種堅持精神依然具有重要意義。

首先，學習上的堅持是成功的基礎。如同荀子所言，即使資質平凡如「駑馬」，只要堅持不懈地努力（「十駕」），也能取得超越天賦的成就。這啟示我們，在面對學習困難時，不應輕易放棄，而應持之以恆。

其次，堅持不僅是個人品質，更是社會進步的動力。許多科學發現和藝術創作都需要長時間的堅持才能取得突破。愛迪生發明電燈泡經歷了上千次失敗，正是「鍥而不舍」精神的體現。

總之，《勤學》中的堅持觀提醒我們，在追求目標的過程中，持之以恆比天賦更重要。我們應培養這種品質，在學習和生活中不斷進步。

---

**題目：「論善用資源」**

荀子指出「君子生非異也，善假於物也」，意思是君子的本性與一般人無異，只是善於借助外物。在資訊發達的今天，我們更應該善用各種資源，如網絡、書籍、師長指導等，來提高學習效率……

**完整參考答案：**
荀子在《勤學》中提出的「君子生非異也，善假於物也」的觀點，對於當代社會具有重要的啟示意義。在資訊時代，善用資源的能力成為成功的關鍵因素。

首先，學習資源的充分利用能顯著提高效率。荀子以「假輿馬者，非利足也，而致千里；假舟楫者，非能水也，而絕江河」為例，說明善用工具可以彌補自身的不足。今天，我們可以通過網絡課程、數字圖書館、學習應用等工具，突破時間和空間的限制。

其次，善用社會資源有助於個人成長。荀子強調的「善假於物」不僅指物質工具，也包括人際資源。向老師請教、與同學討論、參加學術社群等，都是「善假於物」的體現。

然而，善用資源不等於依賴資源。荀子同時強調「用心一也」，提醒我們在使用資源時要保持專注和主動性。只有將外部資源與內在努力結合，才能達到最佳的學習效果。

總之，《勤學》中的資源觀教導我們，在現代社會中，成功不僅取決於個人能力，更取決於善用各種資源的能力。我們應培養這種意識，在學習和工作中充分利用可用資源。
